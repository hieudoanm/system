## 📦 Installation

```bash
curl -fsSL https://github.com/hieudoanm/system/raw/refs/heads/master/packages/cli/bin/system -o system
chmod +x system
sudo mv system /usr/local/bin/
```

## 🚀 Usage

```bash

system version
```
